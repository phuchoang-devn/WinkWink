WinkWink:

Welcome to our project! This application is designed to facilitate online dating interactions between users.
In this group project we had to familarize ourself with Agile Web Development, so our Semester project was to create a WebSite/Service
using Node.js , and inplement best practecis along the creating prosses like: using Docker , requiring Authentication. 
We also decided to user React for the front end.



Features:
User Authentication: Register and login securely.
Profile Management: Create and manage user profiles.
Matching: Find potential matches based on preferences.
Messaging: Communicate with matches via messaging features.
Technologies Used
Node.js
Express.js
MongoDB
React (for frontend)
Docker (for containerization)


Getting Started:
Prerequisites
Node.js installed on your local machine
Docker



Installation:
Clone the repository:[ Link to Git/WinkWink]

Build the Docker images:

docker-compose up build
Run the Docker containers:

docker-compose up

Usage:
Access the application at http://localhost:8080 (if running frontend separately).
API endpoints are available at http://localhost:3000/api.
